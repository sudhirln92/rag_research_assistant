# RAG Research Assistant
The aim of the project is to develop RAG based research Assistant 

## Environment Variables

To configure the project, create a `.env` file in the root directory. Add your Hugging Face API token as shown below:

```shell
HUGGINGFACEHUB_API_TOKEN="your_huggingface_api_token"
```

Replace `"your_huggingface_api_token"` with your actual Hugging Face API token. This token is required to access Hugging Face services.
